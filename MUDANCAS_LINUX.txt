1. 📥 Extração de Protocolos
   → Baixa PDFs do Todoist
   → Extrai números de protocolo

2. 🎯 Lances Automáticos
   → Faz login no Servopa
   → Busca cada protocolo
   → Registra lances automaticamente

3. 💬 Notificações WhatsApp
   → Envia mensagens via Evolution API
   → Confirma sucesso/falha

4. 🔄 Agendamento Inteligente
   → Ciclo Dia 8: primeira tentativa
   → Ciclo Dia 16: segunda tentativa
